Result interpretation

The generated analytics show the downstream orchestrator layer reaching a 54.5k events/s synthetic runtime peak, a 17.6k events/s Redis Streams peak, and a 20.0k events/s pgoutput+Redis peak. These are HybridRuntime processing results after source acquisition; complete-pipeline discussions should use the end-to-end fields in the CSV tables.

The soak and chaos artifacts provide the stability side of the same claim: completed benchmark matrices report zero processing failures, and the published chaos suite recovered all 12 outage/restart scenarios with post-recovery delivery checks.

Published scope

This public snapshot includes generated analytics, CSV data, Markdown reports, suite manifests, and SVG charts. Runtime log files are not published because they contain local filesystem paths from the benchmark host.
