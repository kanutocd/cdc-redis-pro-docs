# cdc-redis-pro-docs

Public documentation mirror for cdc-redis-pro.

The source code lives in the private repository. This repository publishes the
public documentation site for discovery, search, and link sharing.

## Published content

- `docs/index.html` is the landing page.
- `docs/.nojekyll` keeps GitHub Pages from rewriting the static site.
- `docs/robots.txt` allows indexing.

## Deploy

Use GitHub Pages with `docs/` as the publishing source, or deploy with the
included workflow.
